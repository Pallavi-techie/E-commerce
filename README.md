# 🛒 React Authentication-Based E-Commerce Dashboard

A responsive **E-Commerce Dashboard** built using **React** that allows users to register, log in, browse products, manage their cart, and edit their profile. This project was developed as part of a **technical assignment for the Chaintech Network interview process**.

---

## 🚀 Live Demo

🔗 https://ecommerce-dashboard-seven-gamma.vercel.app/

---



## 📌 Project Objective

The goal of this project is to build a **fully functional front-end E-Commerce application with authentication**, protected routes, and responsive UI — **without using a backend**.

Users can:

* Register and login
* Access a protected dashboard
* Browse products
* Add and manage items in cart
* Edit their profile
* Experience a responsive and modern UI

---

## ✨ Features

### 🔐 Authentication

* User **Registration**
* User **Login**
* Authentication using **localStorage**
* **Protected Dashboard Routes**
* Users cannot access the dashboard without login

### 🛍️ Product Dashboard

* View list of products
* Product cards with details
* Responsive product grid layout

### 🛒 Cart System

* Add products to cart
* Remove products from cart
* View cart items

### 👤 Profile Management

* Edit user profile
* Update personal information
* Dynamic profile display

### 📱 Responsive UI

* Mobile-friendly design
* Works across desktop, tablet, and mobile devices
* Light/Dark mode compatibility

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/Pallavi-techie/react-ecommerce-dashboard.git
```

Navigate to project folder:

```
cd react-ecommerce-dashboard
```

Install dependencies:

```
npm install
```

Start development server:

```
npm run dev
```

---

## 🔒 Protected Route Logic

Users must be authenticated to access:

* Dashboard
* Cart
* Profile

If a user is **not logged in**, they will automatically be redirected to the **login page**.
